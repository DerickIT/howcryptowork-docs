# 第十一章：非同质化代币（NFT）

想象一下，花 7000 万美元买一张任何人都可以右键另存为的 JPEG 图片。这听起来荒诞至极，整个前提似乎违反了我们对价值的一切认知：如果一件东西可以以零成本被完美复制，它怎么可能价值数百万？然而，2021 年 3 月，在著名的佳士得拍卖行（Christie's），Beeple 的作品《Everydays》（《每天》）就以这个确切的价格卖给了一位名叫 Metakovan 的买家。他买到的并非独家像素访问权，而是更有意思的东西：一份经过密码学验证、证明他拥有这件"原作"的证书。

## 第一节：数字所有权革命

让那笔 7000 万美元的交易成为可能的，是数字资产运作方式的根本转变。不同于每个单位完全相同的同质化代币，NFT 是非同质化的：每一枚代币都独一无二，由此形成了一个价格发现逐一针对单个资产的市场。这也使得附加在每枚代币上的信息，即所谓的**元数据（metadata）**，显得格外重要。

### NFT 究竟是什么

NFT 出现之前，数字世界有一个根本缺陷：完美可复制性。任何人都可以下载高分辨率图像，创建一份与原件像素级别无法区分的副本。如果副本是免费的，且与原件完全相同，那么任何人怎么可能真正"拥有"一张数字图片？

NFT 通过将数字财产分解成独立的可验证层来解决这个问题。图像文件本身仍可自由复制，但购买 NFT 会获得几个不同的组成部分：

* **代币控制权**：区块链不可篡改地记录了某个持有者控制着 NFT #1234
* **溯源证明**：一份证明该代币源自创作者钱包的证书，确立了真实性
* **使用权**：一份独立的许可协议（通常是链下的），定义了持有者可以如何使用内容
* **功能访问权**：智能合约可以基于代币持有状态授予权限（代币门控功能等）

这些层次可以独立编程，提供了非凡的灵活性。不同于仅仅挂在墙上的画作，NFT 可以随时间演变，在受支持的平台上向创作者输送**版税（royalties）**，与其他数字资产互动，甚至控制它们自己的钱包。一位所有者可能持有一个 NFT，它授予在商业中使用该艺术作品的权利，而图像本身存储在分布式存储网络上，溯源则由创作者的钱包进行锚定。各层保持模块化，并可与其他系统进行组合。

### 独特性是如何实现的

其核心原理其实很简单。ERC-20 等普通代币（第二章引入的同质化代币标准）如同一模一样的美元钞票，而 NFT 则如同有编号的宝可梦卡牌。每枚 NFT 在其智能合约中都会获得一个唯一标识符，区块链维护一个永久账本，记录哪个钱包控制着哪枚代币。不同的 NFT 系列可能存在相同的编号，但系列合约地址与标识符的组合在全球范围内是唯一的，确保了每枚 NFT 在整条区块链上都可以被唯一识别。

一项较新的创新允许 NFT 拥有属于自己的资产。每枚 NFT 都可以与其自己的钱包相关联——这个钱包不是由私钥控制，而是由持有该 NFT 的人控制。这意味着 NFT 可以随着时间的推移积累资产。一个游戏角色 NFT 可以在你游戏过程中收集装备和货币；一个会员 NFT 可以积累活动徽章和奖励。当你出售或转移该 NFT 时，它所持有的一切都会自动随之转移。

然而，这个技术基础引入了每个项目都必须面对的核心设计张力：哪些内容存在链上，哪些存在链下。除技术决策之外，这种灵活性还引入了一个关于所有权如何实际运作的根本性法律空白。

### 版权困境

将权利分解成独立的层次，开辟了重大的法律灰色地带。虽然 NFT 证明了对代币的持有，但底层艺术品的使用权受链下许可和传统版权法的管辖，而这些法律体系往往对去中心化资产的处理并不完善。这些许可在不同司法管辖区的可执行性尚未在法庭上经受过充分检验，持有者到底能拿价值数百万美元的 JPEG 做什么，至今仍是一个悬而未决的问题。

这种模糊性导致了 NFT 世界中的一次重大战略分裂。像无聊猿游艇俱乐部（Bored Ape Yacht Club）这样的项目授予了所有者商业权利，但保留了重要的知识产权控制权。与之形成直接对立的是，一股强大的力量拥抱通过 **Creative Commons Zero (CC0)** 将艺术作品奉献给公共领域。Nouns DAO 和 CrypToadz 等知名项目采用了"版权完全放弃"的方式，允许任何人使用、二次创作和商业化其艺术作品。他们的逻辑是，当一个品牌是开放且无需许可的，就像任何人都可以在其上构建的协议一样，它会变得更有价值。在封闭的、中心化控制的品牌与开放的、去中心化的品牌之间做选择，已成为 NFT 创作者面临的一个根本性的意识形态分叉路口。

这些关于所有权权利的哲学和法律问题之所以重要，正是因为 NFT 是一种复杂的技术产物，其价值完全取决于创作者如何解决这些根本性的基础设施挑战。

## 第二节：超越简单所有权

### 存储方案

创建 NFT 会立即面临一个技术困境：将所有内容存储在链上可以获得最大的永久性，但需要支付高昂的燃气费用；将大部分内容存储在链下则更经济实惠，但多年后 NFT 可能指向已经失效的链接。

大多数项目选择混合方案。区块链记录代币持有权，并包含一个链接，指向一个包含代币名称、描述、图片和属性的文件。持有权注册表是不可篡改的，而实际内容则依赖于外部存储的持续可访问性。

存储方案形成了一个光谱：

* **中心化服务器**：成本最低、最灵活，但如果服务器关闭，NFT 将无法访问
* **IPFS（星际文件系统）**：内容寻址的分布式存储，文件通过其内容哈希进行标识，更难以丢失，但需要持续"钉住（pinning）"才能保持可访问性（第十三章的 DePIN 存储部分也有涉及）
* **Arweave**：一次性付费，通过捐赠基金（即"永久网络"）实现永久存储，前期成本较高
* **链上存储**：最大程度的永久性和抗审查性（如 Autoglyphs），但为单张图像支付燃气费可能高达数千美元

更成熟的 NFT 系列采用分层方式：使用内容寻址 URI（IPFS/Arweave 哈希）确保文件不会被悄悄替换；将关键的溯源信息直接存储在链上；并采用多个钉住服务商作为备份。在可靠的存储基础设施到位后，NFT 便从静态图片演变为可编程的动态资产。

### 高级代币类型

**动态 NFT** 会随时间演变。一张体育卡 NFT 可能会在每场比赛后自动更新球员的数据；数字艺术可能会根据持有者所在城市的天气数据改变颜色；游戏角色积累经验值并升级，其形象和能力随之变化。代币本身成为一个鲜活的、能够响应周围世界的存在。

可组合 NFT 创建资产层次结构：包含其他代币的代币。想象购买一块虚拟世界的土地（一个 NFT），其中包含一栋房子（另一个 NFT），房子里又塞满了家具（更多 NFT）。当持有者出售这块土地时，如果系列设计支持这种嵌套，内部的所有东西可以一起转移。于是出现了与我们对物理房地产的认知相映射的复杂资产树。

半同质化代币模糊了同质化与唯一性之间的界限。活动门票最初可能是完全相同的（同质化的），但一旦使用，就会记录特定的座位、入场时间和活动详细信息，从而变得独一无二。游戏物品在未使用时可能成堆叠放，但一旦被玩家装备使用，便会获得各自独特的历史。

**灵魂绑定代币（SBTs，Soulbound Tokens）**则走向了反方向：它们是专门不可转让的，旨在代表应永久与特定个人绑定的身份、资质、成就或声誉。一个大学学位 NFT 就不应该可以被卖给另一个人。

### NFT 的使用场景类别

**头像类（PFP）项目**代表了早期热潮中主导市场的系列产品，如 CryptoPunks、无聊猿和胖企鹅，作为数字身份符号和社交媒体头像。这些系列经历了爆发式增长，但随着投机热情的降温，也从峰值大幅回落。举例来说，截至 2026 年初，无聊猿的**地板价**（一个系列中最低价 NFT 的价格）已从超过 40 万美元的历史最高点下跌了 90% 以上。

让外界人士一再困惑的问题是：为什么这些数字资产能够命令与豪宅或精美艺术品相当甚至超越的价格？答案部分在于投机，但更多地在于人类社会中根深蒂固的数字部落归属信号需求。在我们日益网络化的世界中，这些数字艺术品的用途远超其视觉吸引力。

就像佩戴劳力士在现实世界中传递成功和社会地位一样，将无聊猿作为你的 Twitter 头像，表明你是一个专属数字社区的成员。这些图像在数字空间中传达着身份、财富和文化认同，而传统地位符号在这里则失去了意义。

价值通过自我强化的网络效应得以积累：当 Jay-Z、塞雷娜·威廉姆斯（Serena Williams）和史蒂芬·库里（Steph Curry）等知名人物采用这些头像时，文化相关性得以放大，带来了主流认知。绝对稀缺性发挥着根本作用，因为每个系列存在的数量总是有上限的。

**生成艺术**：有别于头像类，这一类别的焦点在于由自主系统创作的艺术。Art Blocks 等平台让艺术家编写在铸造时执行的算法，产生独特的、通常颇具复杂性和美学驱动性的输出。Tyler Hobbs 的 *Fidenzas* 或 Snowfro 的 *Chromie Squiggles* 等系列，因其艺术价值、历史意义和算法新颖性而备受推崇，吸引着与社区型头像类截然不同的收藏群体。

**游戏和虚拟世界 NFT**：这些项目代表了区块链游戏中的数字资产，从 *Axie Infinity* 中的生物到 *The Sandbox* 中的土地。虽然"玩赚（play-to-earn）"经济和真正的资产所有权曾是一个强大的叙事，但大多数项目都难以建立可持续的经济模型或在初期投机热情消退后留住玩家。

**功能与访问类 NFT**：这类 NFT 充当数字钥匙，为持有者提供对专属社区、活动、软件或服务的访问权限。它们越来越多地被用于探索忠诚度计划和订阅模式，作为一种可验证且可交易的会员资格证明。

**身份与凭证 NFT**将区块链技术应用于可验证凭证，如文凭、认证或职业执照。不可转让的灵魂绑定 NFT 旨在代表不可让渡的成就或声誉。

尽管存在各种功能价值主张，自 2022 年以来，更广泛的 NFT 市场在交易量和地板价方面都经历了大幅下滑，绝大多数项目难以维持活跃的社区或投机性交易以外的实际效用。整体交易量仍只是巅峰时期的一小部分。但像 CryptoPunks 和无聊猿这样的蓝筹系列，即使在长尾项目归零时，也保留了文化意义和可观的地板价，这印证了 NFT 市场与传统艺术品和收藏品市场的镜像关系：少数几个成熟名字保有价值，而其他大多数则不然。

### 供应机制

项目如何管理供应，对于理解 NFT 价值和市场动态同样至关重要。

**固定供应**制造了绝对的稀缺性。著名的一万个 CryptoPunks 永远不会增加，使得每一个都是已知有限集合的确定份额。

**联合曲线（Bonding curves）**采用算法定价，每次铸造的成本递增。一个系列可能从 0.1 ETH 开始，每次铸造增加 0.01 ETH，因此第 50 个代币的成本为 0.59 ETH。这在奖励早期铸造者的同时，也抑制了后期投机。

**销毁机制**允许代币被永久销毁，制造通缩压力。一些系列使用销毁作为 NFT 进化的方式（销毁三件普通物品来铸造一件稀有物品），或者用来访问专属权益。

## 第三节：技术基础

### ERC-721：规则手册

以太坊上绝大多数 NFT 遵循 ERC-721 标准，该标准定义了独特代币在智能合约层面的运作方式。ERC-20（第二章引入）为每个单位完全相同的同质化代币创建了标准，ERC-721 则为每个都独一无二的非同质化代币做了同样的事情。其核心出奇地简单，仅包含每个 NFT 合约必须实现的几个基本函数：

* `ownerOf(tokenId)`："谁拥有 NFT #1234？"
* `transferFrom(from, to, tokenId)`："将 NFT #1234 从 Alice 转移给 Bob"
* `approve(to, tokenId)`："Alice 授权 Bob 转移她的 NFT #1234"
* `setApprovalForAll(operator, approved)`："Alice 授权市场合约转移她的所有 NFT"

该标准还包括用于元数据（链接到包含名称、描述和图片的 JSON 文件）和枚举（让应用程序发现系列中所有代币）的可选扩展。

### ERC-1155：多代币标准

ERC-721 处理唯一代币，而 ERC-1155 则采取了更灵活的方式。它允许单个智能合约同时管理同质化和非同质化代币，这对于需要独特物品（具有个人历史的传奇武器）和同质化资源（可互换的金币）的游戏生态系统来说尤为强大。

ERC-1155 引入了批量操作：与其为每次代币转移单独提交交易，数十个代币可以在单笔交易中转移，大幅降低燃气费用。这种高效性使其成为需要处理大量多样化资产的区块链游戏和应用的首选标准。

### 安全与常见骗局

这些强大的代币标准实现了精密的功能，但同时也带来了安全漏洞。授权函数（尤其是 `setApprovalForAll`）赋予了广泛的权限，诈骗者积极利用这一点。由于区块链交易不可逆，用户必须时刻警惕各种胁迫：

* **网络钓鱼攻击**：诈骗者伪造官方网站，或在 Discord 和 X（前身为 Twitter）上发送欺骗性链接，诱骗用户将钱包连接到恶意网站，以获取"免费铸造"或"空投"。
* **钱包抽取（Wallet Drainers）**：更高级的骗局会诱骗用户签署一个看似合法的交易（如签名请求），实际上却是一个恶意的有效载荷，它会授予攻击者从受害者钱包中抽走所有有价值资产（NFT 和代币）的权限。

这些风险强调了自我托管的一项核心原则：时刻保持警惕。使用硬件钱包存储高价值资产、使用单独的"临时（burner）"钱包从新项目铸造等最佳实践，已经成为安全导航这一领域的必要条件。

### 发行策略

当项目发行 NFT 时，它们面临着与任何稀缺资源相同的根本挑战：如何公平分配，同时防止机器人和不良行为者主导销售。

发行模式在此背景下不断演进。公平发行向所有人提供同等价格，先到先得；但这通常会被自动化机器人主导。荷兰拍卖以高价开始，价格随时间下降直至需求与供应匹配，这对机器人操纵更具抵抗力。**白名单（Allowlists）**授予预先批准的钱包早期访问权，奖励在公开销售前参与社区建设和互动的用户。一些项目还使用前面描述的联合曲线，价格随每次铸造而上涨。

### Solana NFT：一个平行生态系统

Solana（其架构已在第三章介绍）主要独立于以太坊发展了其自己的 NFT 生态系统，使用不同的标准和工具。Metaplex 框架成为大多数 Solana NFT 的基础，处理元数据、系列和版税配置。较新的标准为 NFT 的转移和使用方式添加了更精密的控制。

市场格局的演变也有所不同。Magic Eden 以用户友好的体验和发行工具主导了早期交易量，而 Tensor 则凭借特征级别竞价和自动定价池等高级功能吸引了职业交易者。与以太坊 OpenSea 长期主导的情况不同，Solana 的 NFT 流动性集中在本土市场。

Solana 的一大独特创新是**压缩 NFT（Compressed NFTs）**（状态压缩已在第三章引入），这使得数百万个 NFT 可以以极低的成本铸造。其原理是仅在链上存储密码学摘要，同时将详细数据保存在链下，从而大幅降低成本。这项技术开启了大规模空投、忠诚度计划和游戏资产等在其他网络上成本过高而无法实现的用例。

版税状况与以太坊的演变相似。市场竞争将创作者费用推向了选项可选，虽然一些 NFT 标准试图在合约层面强制执行版税，但执行仍然依赖于市场的配合。

结合 Solana 的快速执行和低费用，这些功能创造了高速交易文化，价格频繁波动。值得关注的系列包括 Solana Monkey Business、Mad Lads 和 Claynosaurz，每个都体现了 Solana 低成本、试验驱动的文化特色。

## 第四节：NFT 在哪里交易

### 市场战争

NFT 市场最初作为简单的挂单网站出现，但很快演变成了复杂的金融基础设施。OpenSea 通过率先推出并提供最便捷的用户体验而主导了早期市场。然而在 NFT 热潮巅峰期间，OpenSea 变得自满、创新迟缓，为竞争对手创造了机会。

竞争围绕着一个根本性的技术缺陷而激化。NFT 版税从未被纳入 ERC-721 和 ERC-1155 核心代币标准，这意味着创作者费用无法被自动强制执行。虽然较新的 **ERC-2981** 标准允许合约建议版税金额，但实际支付这些版税完全是可选的。这个技术漏洞为市场提供了一种强大的竞争手段：它们可以直接忽略创作者费用。

大多数 NFT 系列将创作者版税设定在 5%-10% 之间，买家传统上在购买价格之外额外支付这笔费用。新兴市场 Blur 看到了利用这一弱点的机会，以三管齐下的策略进入市场：其一，面向职业交易者构建工具，包括高级投资组合管理、实时定价数据流和精密筛选功能；其二，将版税设置为可选，仅要求向创作者支付 0.5% 的最低费用；其三，通过奖励用户 BLUR 代币来激励交易活动。

OpenSea 的回应前后不一、最终收效甚微。它通过其运营商过滤器（2022 年 11 月推出）为较新的系列强制执行全额版税，但无法对旧系列强制收费。交易者自然迁移到了 Blur 成本更低的收费结构，OpenSea 的市场主导地位开始动摇。

聚合协议的兴起使局面愈加复杂。Gem 和 Genie 等平台解决了另一个问题：市场碎片化。它们检查多个市场的价格，并自动在用户获得最佳价格的地方执行交易。这项创新无意间放大了 Blur 的优势。由于聚合商将用户引导至成本最低的市场，Blur 的费用折扣成为了一种结构性优势，吸引了越来越多的交易量。这一基础设施层的价值立竿见影，两家聚合商迅速被相继收购：Gem 被 OpenSea 收购，Genie 被 Uniswap 收购。随着交易活动重新整合回 OpenSea，跨市场路由的必要性降低，聚合商的相关性此后逐渐消退。

Blur 的策略最初奏效了。到 2023 年 2 月，Blur 的交易量超过了 OpenSea。到 2023 年 8 月，这场竞争迫使 OpenSea 完全放弃了版税执行政策。但 Blur 的主导地位被证明是短暂的。其大部分交易量是由代币激励和空投挖矿驱动的，而非有机需求。随着激励措施枯竭、NFT 交易量整体崩溃，Blur 的结构性优势变得无足轻重。到 2025 年，OpenSea 重新夺回了以太坊 NFT 主要市场的地位，这一复兴在 Blur 鼎盛时期几乎无人预料。

### 定价机制

理解 NFT 的定价需要掌握几个与传统市场不同的关键概念。

任何 NFT 系列中最受关注的指标是地板价。它作为系列的基准估值，但可能具有很大的误导性。带有稀有特征的 NFT 可能以地板价的 10 倍以上成交。例如，拥有金色皮草和激光眼的无聊猿，比拥有普通棕色皮草和正常眼睛的那只价值高得多。

这种价值差异催生了对更精密定价方法的需求。**特征定价（Trait-based pricing）**作为一种解决方案应运而生，它考虑到每个 NFT 的个别特征，而不是将系列中的所有作品视为等同。

另一项创新——系列范围竞价（Collection-wide bidding）——解决了一个不同的问题：流动性不足。买家不必竞购某一特定的 NFT，而是可以对任何满足特定标准的 NFT 进行出价，例如"任何拥有激光眼的无聊猿"。这改善了卖家的流动性，使价格发现更有效率。代价在于哲学层面：它将理论上独特的资产商品化了。Blur 曾试图通过给特征级别竞价奖励积分来推广这种方式，OpenSea 也增加了对系列和特征报价的支持，但实际上特征竞价从未获得广泛采用。大多数交易活动仍然集中在扫楼（floor sweeps）和系列范围竞价上，而非细粒度的特征策略。

唯一性与可替代性之间的这种张力，正是 NFT 的核心所在。每一个设计决策，从市场功能到定价机制，都反映了这个根本性的悖论，并决定了 NFT 的使用方式和价值评估。
